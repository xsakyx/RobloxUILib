# V6.2 — Harmony Pass

## Goal

Turn the V6.1 visual improvement into a completed interaction system: responsive under scaling, recoverable, customizable, documented, and pleasant across device sizes.

## Included

- Color preview moved below H/S/V sliders with readable contrast and optional hex copy.
- Effective-viewport responsive sizing and forced page scrolling on narrow/touch layouts.
- Dense-height behavior that protects navigation before optional profile/search content.
- Ten-second UI scale preview with Keep, Revert, timeout recovery, and a 100% reset action.
- Recoverable window dragging so the interface cannot be permanently lost off-screen.
- Custom Roblox image IDs for window, settings, tab, and button icons.
- Roblox avatar/profile surface with custom image and runtime update support.
- Official Infinite Yield launcher with confirmation and failure feedback.
- New Aurora and Ember theme presets plus additional visual polish.
- Obsidian-ready project memory and release gate documentation.

## Validation required before publish

- Luau syntax for all entrypoints and Showcase.
- Byte-identical compatibility files.
- Static checks for recovery, responsive, icon, avatar, and integration contracts.
- In-Roblox visual QA using the Showcase on desktop and a narrow window.

## Known boundary

Automated local tooling can validate structure and syntax, but final Roblox rendering and input behavior require an in-client test.
