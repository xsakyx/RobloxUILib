# V6.3 — Harmony

V6.3 turns RenLib's V6.2 polish into a clearer composition system.

## Visible changes

- Sections now contain a distinct inner surface so cards and controls read as separate hierarchy levels.
- Navigation supports category labels that disappear cleanly in compact mode.
- Three-stop accent gradients improve depth without depending on excessive glow.
- New original palettes: Prism Frost, Moss Archive, and Velvet Latte.
- Optional Frosted material mode with adjustable blur and mobile limits.
- Responsive dashboard hero, live clock, avatar, metric cards, and quick actions.
- Nested controls with automatic height propagation.
- Dedicated multi-select dropdown API with selection names, checks, clear, and select-all helpers.

## Completion work

- Frosted blur is disabled when minimized and destroyed on unload.
- Material mode can switch live without restarting the window.
- Dashboard cards retain reading order in a single-column viewport.
- Nested dropdown and color-picker expansion updates the parent and page canvas.
- Legacy `Starlight` theme calls map to the original `Celestial` palette for compatibility; the copied name is no longer shown in the interface.
- Reference screenshots and design reasoning are saved in [[Design/Harmony References]].
